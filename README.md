This is the CAD, budget, schematics, and any other additional files for a 6-DOF robot arm that looks for and picks up specific candy types in a Halloween candy bowl corresponding to user input.
I made this project both as part of Hack Club Stasis and as a personal passion project. First, it would be sick if kids rang the doorbell on Halloween and a robotic arm gave the candy to them instead of an actual person. Also, even when it fully functions in giving candy, I can repurpose to do other things, like stack objects, draw patterns onto paper, and more. Hack Club will give me both the money to build this project and Open Sauce tickets when it is finished.
